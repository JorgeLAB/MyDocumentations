Express

Depois de ter realizado a instação do NodeJS iremos fazer a instalação do express em nosso terminal.

Framework web rápido, flexível e minimalista Node.js | Site: Express

$ npm install express --save

Realizada a instalação do express podemos estar o executando realizando a instalação do template ejs | Site: EJS

O EJS junta HTML e JS de forma a ser executádo em um único arquivo mesclado.

$ express create backend --ejs

Vamos realizar: 

$ cd backend && npm install

Pronto foi realizada a instação das dependências

Repare nos arquivos gerados:

  • app.js
  • bin
  • node_modules
  • package.json
  • package-lock.json
  • public
  • routes
  • views

Gostaria que repara-se neste arquivo destacado é o principal, onde há os requires e inicio de seu programa.

Em bin/www você encontrará as constifurações de rotas e portas. WebSever.

Execute:

$ node ./bin/www
  • app.js
  • bin
  • node_modules
  • package.json
  • package-lock.json
  • public
  • routes
  • views

O server será carregado mas podemos abrir o arquivo package.json e reparar o seguinte: 

"scripts": {
  "start": "node ./bin/www"
}

Além de podemos executar o server com $ node ./bin/www podemos executá-lo com $ npm start.
